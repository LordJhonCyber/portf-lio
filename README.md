# Portfólio Futurista — Jhon Wictor

Este pacote contém um site com visual futurista (neon/cyber) focado em Tecnologia e Desenvolvimento, incluindo um GIF animado em `assets/tech-hero.gif`.

## Publicar no GitHub Pages
1. Crie um repositório (ex.: `portfolio`), faça upload de todos os arquivos.
2. Em **Settings → Pages**, selecione a branch `main` e a pasta **/root** e salve.
3. O site ficará disponível em `https://SEU_USUARIO.github.io/portfolio/`.

## Personalizar
- Substitua `assets/tech-hero.gif` por um GIF próprio, mantendo o nome do arquivo, ou ajuste o caminho no `index.html`.
- Edite textos, projetos e links no `index.html`.
- Ajuste cores e efeitos no `styles.css`.
